# KalindarCli

Query ICS files.

## Installation

Add this line to your application's Gemfile:

    gem 'kalindar_cli'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kalindar_cli

## Usage

    query_ics <path_to_ics_file> [number_of_days]
